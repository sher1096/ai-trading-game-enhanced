# AITradeGame UI 重构方案

## 当前问题分析

### Header区域
- ❌ 按钮过多且拥挤（6个按钮排成一排）
- ❌ 缺少功能分组和视觉层次
- ❌ 渐变色按钮与整体风格不协调
- ❌ 主要操作和次要操作混在一起

### 整体布局
- ⚠️ 缺少顶部导航栏
- ⚠️ 功能入口不够清晰

## 改进方案

### 1. Header 重构

#### 左侧保留
- AITradeGame 标题
- 运行状态指示器
- GitHub 链接

#### 右侧重新组织（从左到右）
1. **快捷工具组**
   - 刷新按钮（图标）

2. **主导航区**（导航风格）
   - 仪表盘（新增，回到主页）
   - 币种管理
   - 实盘交易

3. **主操作**
   - 添加模型（突出的主按钮）

4. **系统菜单**（下拉菜单）
   - 设置
   - API提供方

### 2. 样式优化

#### 导航按钮样式
- 去除花哨渐变
- 统一使用清爽的扁平风格
- 激活状态使用底部边框高亮
- Hover效果使用背景色变化

#### 按钮分组
- 使用视觉分隔线
- 不同功能组有明确间距

### 3. 新增功能

#### 面包屑导航（可选）
- 显示：首页 / 币种管理 / 详情
- 帮助用户了解当前位置

#### 通知中心（未来）
- 系统通知
- 交易提醒

## 实施步骤

1. 修改 index.html Header部分
2. 更新 style.css 按钮和导航样式
3. 添加JavaScript交互逻辑
4. 测试响应式布局
