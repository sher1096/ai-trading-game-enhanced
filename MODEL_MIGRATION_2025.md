## 🔧 如何更新你的配置

### 方法1: 更新API提供方设置

1. 登录 http://localhost:5000
2. 点击 "API提供方" 管理
3. 编辑现有配置，更新模型名称：

**OpenAI 提供方**:
```
旧配置:
模型: gpt-4o

新配置:
模型: gpt-5
```

**Claude 提供方**:
```
旧配置:
模型: claude-3-5-sonnet-20241022

新配置:
模型: claude-sonnet-4-5
```

### 方法2: 创建新的API提供方

推荐直接创建新的提供方使用2025模型：

**OpenAI GPT-5**:
```
名称: OpenAI-GPT5-2025
API URL: https://api.openai.com/v1
API Key: sk-[你的密钥]
模型: gpt-5, gpt-5-mini, gpt-5-nano
```

**Claude 4.5**:
```
名称: Claude-4-5-2025
API URL: https://api.anthropic.com/v1
API Key: sk-ant-[你的密钥]
模型: claude-sonnet-4-5, claude-opus-4-1, claude-haiku-4-5
```
